### 🚀 Featured Project

#### [⚡ API Studio](https://github.com/openAPI-Studio/open-post)
A fully offline API client for VS Code, CLI, and Desktop.

* **Protocols**: REST, GraphQL, WebSocket, and gRPC.
* **Offline & Secure**: No cloud accounts, no telemetry, and a built-in AES-256-GCM encrypted secret vault.
* **Git-Friendly**: Stores collections in plain JSON for easy version control.
* **Mock & MCP**: Multi-instance mock servers and Model Context Protocol (MCP) server integration to power AI clients.
